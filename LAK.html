A work-in-progress diplomatic edition of Deimel’s Liste der archaischen Keilschriftzeichen von Fara, based on the Oracc-LAK font by Steve Tinney.

Missing text that is clearly required (e.g., unclosed parentheses, or the sign number in the reference to 𒅔 just below n. 132) is added with green highlighting like this. Any other corrections or remarks are written in smaller type after each entry.

A version is available that replicates the layout of the original, including page and line breaks. For ease of reading on screens of various sizes, the text can also be reflowed, eliminating page breaks and using automatic line breaking. The reflowed version is the default on narrow viewports; the original layout is the default on wider ones. The text can also viewed side-by-side with a facsimile for proofreading. The checkboxes below, as well as URL query parameters, control these options.

Each sign number can be used as a fragment identifier (#𝑛 for sign 𝑛), as well as each page number (#p𝑛 for page 𝑛). References to sign numbers in the text are linked to the respective signs. Artefact designations are linked to CDLI; references to locations within the artefact are linked to the appropriate location in an Oracc transliteration (such links are green).

The Neo-Assyrian font is 𒀭𒀝𒌷𒀏𒄴𒋢𒊻. The Latin font is STIX Two by Ross Mills, John Hudson, and Paul Hanslow.
